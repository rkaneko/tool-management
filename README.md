# Tool Management

## Purposes

- Manage CLIs declaratively.
- Share CLIs across devices which I use for software development.

## Prerequisites

- Env: Ubuntu 24.04
- Required tools:
  - Homebrew: https://brew.sh/

```bash
$ brew install aqua

$ aqua -v
aqua version 2.48.1 (Homebrew)

# Install tools using aqua
$ aqua i
```
